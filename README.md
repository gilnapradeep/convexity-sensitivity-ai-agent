# Data Analyst Convexity Sensitivity AI Agent

This project analyzes bond portfolio duration, convexity, DV01, yield curve sensitivity, Monte Carlo VaR, and machine learning-based bond risk prediction.

## Dataset Overview

- Bond portfolio data: 300 bonds with pricing, duration, convexity, DV01, rating, sector, and maturity details.
- Yield curve history: INR yield curve data across multiple tenors.
- Monte Carlo scenarios: 1000 simulated interest rate scenarios with P&L impact.
